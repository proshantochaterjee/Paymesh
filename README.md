# WorkforceOS

WorkforceOS is a programmable workforce finance platform on Stellar Testnet, featuring advanced smart contract architecture, full-stack Next.js and NestJS implementation, event streaming, and enterprise-grade security.

## Features
- **Programmable Treasury**: Role-based access control, deposit/withdraw flows.
- **Employee Registry**: On-chain and off-chain sync, dual-phase creation.
- **Payroll Engine**: Chunked batch payroll execution with partial failure handling.
- **Milestone Engine**: Contractor payment state machine (Draft -> Funded -> Approved -> Released).
- **Event Streaming**: BullMQ-backed Stellar RPC indexing for real-time Postgres projections.

## Architecture
- **Contracts**: Stellar Soroban (Rust).
- **Backend**: NestJS, Prisma (PostgreSQL).
- **Frontend**: Next.js 15 App Router, Tailwind CSS, shadcn/ui.
- **Auth**: Better Auth + Stellar Freighter Wallet challenge/response.
- **CI/CD**: GitHub Actions workflows for testing and deployment.

## Technology Stack
- Rust (wasm32v1-none)
- Node.js 20, TypeScript
- PostgreSQL, Redis
- Vercel (Frontend), Render/Railway (Backend)

## Installation Guide
1. Run `npm install` to install dependencies.
2. Initialize PostgreSQL and Redis locally (or via Docker).
3. Setup environments by copying `.env.example` to `.env` in `apps/backend` and `apps/frontend`.
4. Run migrations: `npx turbo run generate` followed by `npx prisma migrate dev` in backend.
5. Start the platform: `npx turbo run dev`.

## Smart Contract Deployment Guide
Run `scripts/deploy-contracts.sh` to deploy to Stellar Testnet. 
The deployer address must be funded on Testnet. Ensure the Stellar CLI is installed.
The deployed addresses are saved to `deployed-addresses.testnet.json`.

## Event Streaming Architecture
The `indexer` module polls Stellar RPC for contract events, filtering by relevant topics, and updates `Transaction` and `Milestone` records in Postgres using idempotent processing.

## Testing Instructions
- Contracts: `cargo test --all`
- Backend: `npx vitest` inside `apps/backend`
- Frontend: Next.js tests
- Monorepo full suite: `npx turbo run test`
